# Spotify Clone — UI

Recriação da interface do Spotify desenvolvida com Next.js e Tailwind CSS. O projeto tem como objetivo praticar componentização, layout com Flexbox/Grid e estilização utilitária com Tailwind.

## Stack

- **Next.js 14** com App Router
- **TypeScript**
- **Tailwind CSS**
- **Lucide React** para ícones

## Pré-requisitos

- Node.js 18+

## Instalação

```bash
git clone https://github.com/rafaelAmora/spotify-clone.git
cd spotify-clone
npm install
npm run dev
```

## Funcionalidades

- Sidebar com navegação (Home, Search, Your Library) e playlists
- Grid de álbuns recentes com botão de play ao hover
- Seção "Made for you" com cards de Daily Mix
- Footer com player: controles de reprodução, barra de progresso e controle de volume

## Scripts

| Comando | Descrição |
|---------|-----------|
| `npm run dev` | Inicia o servidor de desenvolvimento |
| `npm run build` | Gera o build de produção |
| `npm run start` | Inicia o servidor de produção |
| `npm run lint` | Executa o linter |
